Non-Production API

This API is for testing and development purposes only. Use it to experiment!

You can send 10 connections every 3 seconds.

Available Endpoints

GET /api/utc

Returns the current UTC time in JSON format.

Parameters:
  • None
Returns:
{ "utc": "2025-06-06T12:34:56Z" }
GET /api/random/password/[length]

Generates and returns a random password in JSON format.

Parameters:
  • length (optional, integer): Length of the password generated. Defaults to 16.
Returns:
{ "password": "c8rGxl6O7ghrZiPo" }
Note: This API is not for production use. Data may be reset without notice.
