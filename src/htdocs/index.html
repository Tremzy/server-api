Non-Production API

This API is for testing and development purposes only. Use it to experiment!

You can send 5 connections every 5 seconds.

Available Endpoints

GET /api/utc

Returns the current UTC time in JSON format.

Parameters:
  • None
Returns:
{ "utc": "2025-06-06T12:34:56Z" }
GET /api/users

Retrieves a list of users.

Parameters:
  • limit (optional, integer): Maximum number of users to return.
  • offset (optional, integer): Number of users to skip.
Returns:
[ { "id": 1, "name": "Alice", "email": "alice@example.com" }, { "id": 2, "name": "Bob", "email": "bob@example.com" } ]
POST /api/login

Authenticate a user and receive a token.

Parameters (JSON body):
{ "username": "your_username", "password": "your_password" }
Returns:
{ "token": "abcdef1234567890" }
Note: This API is not for production use. Data may be reset without notice.
